## Observações

O template não segue todas as regras do manual.

- O espaçamento entre as linhas é diferente do espaçamento do manual.

Espaçamento entre linhas é complexo. O manual faz referência ao
espaçamento do Microsoft Word, no entanto a forma como o
Typst faz o cáculo é diferente. Assim, foi feita uma aproximação.

- A fonte é diferente da fonte do manual.

Arial é a fonte definida no manual. Essa fonte não é livre.
É possível utilizar a fonte TeX Gyre Heros. Essa é uma
família de fontes que se assemelha à fonte Arial.

- Tabelas tem formatação diferente das tabelas do manual.

O manual define brevemente a formatação de tabelas.
Além disso, faz referência ao modelo do IBGE.
Por enquanto, as tabelas não terão um formato definido.
